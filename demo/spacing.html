Spacing Utilities

Margin and padding utilities for consistent spacing.

Margin Classes

.m-1 (margin all sides)
.mt-3 (margin top)
.mb-4 (margin bottom)
.ml-2 (margin left)
.mr-2 (margin right)

Padding Classes

.p-1 (padding all sides)
.pt-3 (padding top)
.pb-4 (padding bottom)
.pl-5 (padding left)
.pr-5 (padding right)

Spacing Scale

Available spacing values: 0, 1, 2, 3, 4, 5

  • 0 = 0
  • 1 = 0.25rem (4px)
  • 2 = 0.5rem (8px)
  • 3 = 1rem (16px)
  • 4 = 1.5rem (24px)
  • 5 = 3rem (48px)
