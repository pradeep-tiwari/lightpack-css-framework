Tooltips

Contextual information that appears on hover.

Button Tooltips

Hover over buttons to see tooltips.

Text Tooltips

Tooltips can be applied to any element.

Hover over this text to see a tooltip.

Tooltips provide additional context for UI elements.

Icon Tooltips

Perfect for icon buttons and actions.

HTML Structure

<!-- Basic Tooltip -->
<button class="btn btn-primary tooltip" data-tooltip="Tooltip text!">
  Hover me
</button>

<!-- Text Tooltip -->
<span class="tooltip" data-tooltip="Additional info">
  hover this text
</span>

Usage

  • Add .tooltip class to any element
  • Use data-tooltip="Your text" attribute for the tooltip content
  • Tooltips appear on hover automatically
  • Works with buttons, text, icons, and any HTML element
  • Pure CSS - no JavaScript required
