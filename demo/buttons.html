Buttons

Versatile button components with multiple variants, sizes, and states.

Solid Buttons

Default button style with solid backgrounds.

<button class="btn btn-primary">Primary</button>
<button class="btn btn-secondary">Secondary</button>
<button class="btn btn-success">Success</button>

Outline Buttons

Buttons with transparent backgrounds and colored borders.

<button class="btn btn-primary btn-outline">Primary</button>
<button class="btn btn-success btn-outline">Success</button>

Ghost Buttons

Minimal buttons with no border, perfect for tertiary actions.

<button class="btn btn-primary btn-ghost">Primary</button>
<button class="btn btn-danger btn-ghost">Danger</button>

Button Sizes

Three size variants: small, default, and large.

<button class="btn btn-primary btn-sm">Small</button>
<button class="btn btn-primary">Default</button>
<button class="btn btn-primary btn-lg">Large</button>

Disabled State

Buttons can be disabled using the disabled attribute or .disabled class.

<button class="btn btn-primary" disabled>Disabled</button>
<button class="btn btn-secondary disabled">Disabled</button>
