Key Features

  • Shadcn-inspired Design: Modern black/white primary colors that adapt beautifully to light and dark modes
  • Fully Responsive: Mobile-first approach with breakpoints for all screen sizes
  • Dark Mode Ready: Toggle between light and dark themes seamlessly
  • Semantic Variables: CSS custom properties for easy customization
  • Lightweight: Minimal footprint with maximum functionality
  • No Dependencies: Pure CSS and vanilla JavaScript
