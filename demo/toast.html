Toast Notifications

Temporary notification messages that appear and auto-dismiss.

Toast Variants

Click buttons to show toast notifications in different positions.

Toast Positions

Toasts can appear in four corner positions.

Sticky Toast

Toasts that don't auto-dismiss (must be closed manually).

JavaScript API

// Show a toast notification
Lightpack.showToast({
  message: 'Your message here',
  type: 'success',        // 'success', 'danger', 'warning', 'info'
  position: 'bottom-right', // 'bottom-right', 'bottom-left', 'top-right', 'top-left'
  sticky: false            // true = doesn't auto-dismiss
});

// Remove a specific toast
Lightpack.removeToast(toastId, position);

Features

  • Auto-dismisses after 5 seconds (unless sticky)
  • Manual close button on each toast
  • Four corner positions
  • Multiple toasts stack vertically
  • Smooth fade-in animation
  • Accessible with ARIA attributes
