Introduction

In today's fast-paced web development landscape, having the right tools can make all the difference. Lightpack CSS is a modern, utility-first CSS framework that empowers developers to build beautiful, responsive web applications with minimal effort.

Unlike traditional CSS frameworks that come with pre-designed components, Lightpack provides you with low-level utility classes that let you build completely custom designs without ever leaving your HTML.

Getting Started

Getting started with Lightpack is incredibly simple. You can include it via CDN or install it through npm.

CDN Installation

Add these lines to your HTML file:

<link rel="stylesheet" href="https://cdn.jsdelivr.net/npm/@lightpackcss/framework/dist/lightpack.min.css">
<script src="https://cdn.jsdelivr.net/npm/@lightpackcss/framework/dist/lightpack.min.js"></script>

NPM Installation

Or install via npm for build tool integration:

npm install @lightpackcss/framework

Core Concepts

Lightpack is built around several core concepts that make it powerful yet easy to use.

Utility-First Approach

Instead of writing custom CSS, you compose designs using utility classes. This approach leads to faster development and more consistent designs. 

<div class="flex items-center justify-between p-4 bg-surface radius-md">
  <h3 class="fs-lg fw-700">Title</h3>
  <button class="btn btn-primary">Action</button>
</div>

Responsive Design

Every utility class can be applied conditionally at different breakpoints, making responsive design a breeze. 

<div class="grid grid-cols-1 grid-cols-md-2 grid-cols-lg-4 gap-4">
  <!-- Content -->
</div>

Best Practices

To get the most out of Lightpack, follow these best practices:

1. Use the Spacing Scale

Lightpack provides a consistent spacing scale from --g-1 to --g-9. Use these values for margins, padding, and gaps to maintain visual consistency.

2. Leverage CSS Variables

All colors, typography, and spacing use CSS variables, making theming and customization trivial. 

:root {
  --color-primary: #6366f1;
  --radius-md: 12px;
}

3. Mobile-First Responsive Design

Always design for mobile first, then add responsive variants for larger screens using -sm, -md, -lg, and -xl suffixes.

Conclusion

Lightpack CSS provides everything you need to build modern, responsive web applications. Its utility-first approach, combined with powerful components and excellent customization options, makes it an ideal choice for projects of any size.

Whether you're building a simple landing page or a complex web application, Lightpack has you covered. Start using it today and experience the difference!
