# Lightpack CSS Framework - Demo Suite

## Overview
This demo suite showcases all components, layouts, and utilities available in the Lightpack CSS Framework. Each HTML file demonstrates a specific feature with real-world examples and code snippets.

## How to Use

1. **Open any demo file** in your browser
2. **Click "Toggle Theme"** button to switch between light and dark modes
3. **Navigate** using the sidebar to explore different components
4. **View code examples** in the gray code blocks
5. **Inspect real-world examples** to see components in context
