When to Use Each Badge

Success Active/Completed States

Use for positive status, completed tasks, active states.

Active Completed Published Verified Approved

Danger Error/Inactive States

Use for errors, failed states, inactive status, critical issues.

Inactive Failed Blocked Overdue Deleted

Warning Pending/Caution States

Use for pending actions, warnings, moderate priority items.

Pending Draft Review Expiring Limited

Info Informational States

Use for neutral information, system messages, new features.

New Beta Updated Info Preview

Primary Important/Featured

Use for important items, featured content, high priority (non-status).

Featured Premium VIP Hot Popular

Secondary Categories/Types

Use for secondary categories, types, less important classifications.

Standard Regular General Basic

Muted Low Priority/Archived

Use for archived items, deprecated features, minimal attention needed.

Archived Legacy Deprecated Old
