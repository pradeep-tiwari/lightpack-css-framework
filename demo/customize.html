Customization Guide

Learn how to customize fonts, colors, and themes in Lightpack CSS.

1. Custom Font Family

Change the default font to match your brand.

Using Google Fonts

<!DOCTYPE html>
<html lang="en">
<head>
  <!-- 1. Load Google Font -->
  <link rel="preconnect" href="https://fonts.googleapis.com">
  <link rel="preconnect" href="https://fonts.gstatic.com" crossorigin>
  <link href="https://fonts.googleapis.com/css2?family=Inter:wght@300;400;500;600;700&display=swap" rel="stylesheet">
  
  <!-- 2. Load Lightpack CSS -->
  <link rel="stylesheet" href="lightpack.css">
  
  <!-- 3. Override font variable -->
  <style>
    :root {
      --font-family: 'Inter', system-ui, sans-serif;
    }
  </style>
</head>
<body>
  <!-- Your app content -->
</body>
</html>

Popular Font Combinations

<!-- Modern & Clean -->
:root {
  --font-family: 'Inter', system-ui, sans-serif;
}

<!-- Professional -->
:root {
  --font-family: 'Roboto', system-ui, sans-serif;
}

<!-- Elegant -->
:root {
  --font-family: 'Poppins', system-ui, sans-serif;
}

<!-- Geometric -->
:root {
  --font-family: 'Montserrat', system-ui, sans-serif;
}

<!-- Tech/Startup -->
:root {
  --font-family: 'DM Sans', system-ui, sans-serif;
}

2. Custom Font Sizes

Adjust typography scale to match your design.

Method 1: Scale Everything (Recommended)

<style>
  <!-- Change base size - all rem units scale automatically -->
  html {
    font-size: 18px;  /* Default is 16px */
  }
</style>

<!-- Result: -->
<!-- 1rem = 18px (instead of 16px) -->
<!-- 1.5rem = 27px (instead of 24px) -->
<!-- Everything scales proportionally! -->

Method 2: Override Specific Sizes

<style>
  :root {
    /* Override individual font sizes */
    --fs-xs: 0.875rem;    /* 14px */
    --fs-sm: 1rem;        /* 16px */
    --fs-base: 1.125rem;  /* 18px - Body text */
    --fs-md: 1.25rem;     /* 20px */
    --fs-lg: 1.5rem;      /* 24px */
    --fs-xl: 1.75rem;     /* 28px */
    --fs-2xl: 2rem;       /* 32px */
    --fs-3xl: 2.5rem;     /* 40px */
    --fs-4xl: 3rem;       /* 48px */
    --fs-5xl: 3.75rem;    /* 60px */
  }
</style>

Method 3: Change Body Text Only

<style>
  :root {
    --fs-base: 1.125rem;  /* 18px body text */
  }
</style>
âœ… Best Practice
Use Method 1 (scale HTML font-size) for consistent proportions across your entire app.

3. Custom Color Theme

Create your own color scheme by overriding CSS variables.

Basic Theme Override

<style>
  :root {
    /* Primary color (buttons, links, accents) */
    --color-primary: #4f46e5;
    --color-primary-light: #6366f1;
    --color-primary-dark: #4338ca;
    
    /* Status colors */
    --color-success: #10b981;
    --color-danger: #ef4444;
    --color-warning: #f59e0b;
    --color-info: #3b82f6;
  }
</style>

Complete Brand Theme Example

<style>
  :root {
    /* Brand Colors */
    --color-primary: #6366f1;        /* Indigo */
    --color-primary-light: #818cf8;
    --color-primary-dark: #4f46e5;
    
    /* Background Colors */
    --color-background: #f9fafb;     /* Light gray */
    --color-surface: #ffffff;        /* White */
    --color-border: #e5e7eb;         /* Subtle border */
    
    /* Text Colors */
    --color-text: #111827;           /* Dark gray */
    --color-text-muted: #6b7280;     /* Medium gray */
  }
  
  /* Dark Theme Override */
  .theme-dark {
    --color-primary: #818cf8;        /* Lighter indigo for dark mode */
    --color-background: #0f172a;     /* Dark blue */
    --color-surface: #1e293b;        /* Slate */
    --color-border: rgba(255,255,255,0.1);
    --color-text: #f1f5f9;           /* Light text */
    --color-text-muted: #94a3b8;     /* Muted light */
  }
</style>

4. Dark Mode Setup

Enable dark mode toggle in your app.

Toggle Dark Mode

<!-- Add toggle button -->
<button onclick="Lightpack.toggleTheme()">
  Toggle Dark Mode
</button>

<!-- Or manually toggle -->
<button onclick="document.body.classList.toggle('theme-dark')">
  Toggle Dark Mode
</button>

Default to Dark Mode

<!-- Add class to body tag -->
<body class="theme-dark">
  <!-- Your content -->
</body>

Auto-detect System Preference

<script>
  // Detect system dark mode preference
  if (window.matchMedia('(prefers-color-scheme: dark)').matches) {
    document.body.classList.add('theme-dark');
  }
</script>

5. Complete Customization Example

Putting it all together - custom font, sizes, and colors.

<!DOCTYPE html>
<html lang="en">
<head>
  <meta charset="UTF-8">
  <meta name="viewport" content="width=device-width, initial-scale=1.0">
  <title>My Custom App</title>
  
  <!-- 1. Load Custom Font -->
  <link rel="preconnect" href="https://fonts.googleapis.com">
  <link href="https://fonts.googleapis.com/css2?family=Inter:wght@400;500;600;700&display=swap" rel="stylesheet">
  
  <!-- 2. Load Lightpack CSS -->
  <link rel="stylesheet" href="lightpack.css">
  
  <!-- 3. Custom Overrides -->
  <style>
    /* Scale everything proportionally */
    html {
      font-size: 18px;
    }
    
    /* Custom font and colors */
    :root {
      /* Font */
      --font-family: 'Inter', system-ui, sans-serif;
      
      /* Brand Colors */
      --color-primary: #6366f1;
      --color-primary-light: #818cf8;
      --color-primary-dark: #4f46e5;
      
      /* Backgrounds */
      --color-background: #f9fafb;
      --color-surface: #ffffff;
    }
    
    /* Dark Theme */
    .theme-dark {
      --color-primary: #818cf8;
      --color-background: #0f172a;
      --color-surface: #1e293b;
    }
  </style>
  
  <!-- 4. Load Lightpack JS -->
  <script src="lightpack.js"></script>
</head>
<body>
  <!-- Your app content -->
  <button class="btn btn-primary" onclick="Lightpack.toggleTheme()">
    Toggle Dark Mode
  </button>
  
  <script>
    // Initialize Lightpack
    Lightpack.init();
  </script>
</body>
</html>
